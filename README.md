# FazoPix Overlay

O **FazoPix Overlay** mostra as trolagens das suas doações direto na tela do
seu jogo, enquanto você joga ao vivo. 🎮

## ⬇️ Baixar

### [» Baixar o FazoPix Overlay para Windows «](https://github.com/codevops/fazopix.gg-app-overlay/releases/latest/download/fazopix-overlay-setup.exe)

## Como instalar e usar

1. Baixe e abra o instalador
2. Abra o **FazoPix Overlay** — o atalho fica na área de trabalho
3. Cole o link do seu overlay de trolagem (você encontra no painel da FazoPix,
   em **https://fazopix.gg/settings?tab=trolagens**)
4. Escolha em qual monitor a trolagem deve aparecer. (escolha o monitor que você joga)
5. Pronto! Quando alguém doar uma trolagem, ela aparece sobre o seu jogo 🎉

> 💜 Dica: jogue em modo **janela** (borderless / janela sem borda) para a
> trolagem aparecer por cima. O app fica discreto na bandeja do Windows e se
> atualiza sozinho.

## Apareceu "O Windows protegeu seu computador"?

É normal e o app é seguro. Clique em **Mais informações** e depois em
**Executar assim mesmo** para continuar a instalação.
